* [Show package details](https://uappexplorer.com/snap/ubuntu/gradio)

## Install
* _snap install gradio_
* _snap install gnome-3-24_
* _snap connect gradio:gnome-3-24-platform gnome-3-24:gnome-3-24-platform_
* _snap run gradio_
